# Wordlist Generator

**[English Version](README.md)**

Gerador de wordlists personalizadas baseadas em OSINT e Engenharia Social.

### Funcionalidades:
- **Input Biográfico:** Coleta de dados públicos (familiares, pets, datas).
- **Concatenação:** Combina termos com números registrados.
- **Leet Speak:** Mutação básica de caracteres (ex: a -> 4).
- **Output:** Arquivo `wordlist.txt` pronto para uso.

### Uso:

```
python main-pt.py
```
